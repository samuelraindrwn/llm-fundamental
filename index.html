ğŸ”¢ Ubah Huruf Jadi Angka

Step 1: Komputer itu buta huruf, taunya cuma angka. Jadi kita harus kasih nomor absen (index) buat tiap huruf.

Nama kerennya: Tokenization!

ğŸ“– Gimana Caranya?

  1. Ambil semua karakter dari teks
  2. Hapus duplikat (ambil karakter unik saja)
  3. Urutkan sesuai abjad A sampai Z
  4. Berikan nomor urut mulai dari 0

# 2. Bikin "kamus" huruf â†’ angka

ğŸ“Š Bikin Angka Menjadi Lebih Bermakna (Embedding)

Masalah fatalnya bro: Kalo kita cuma kasih angka (misal 'a'=2, 'b'=3), komputer bakal mikir 'a' sama 'b' itu mirip karena angkanya deketan.

Padahal 'Apel' sama 'Batu' kan beda jauh! Jadi angka doang GAK CUKUP.

Solusinya: Kita ubah setiap huruf menjadi vektor (deretan angka). Dengan demikian, tiap huruf memiliki representasi numerik yang lebih kaya.

âŒ Sebelum "a" â†’ 2 Hanya 1 angka, informasi terbatas
â†’
âœ… Sesudah "a" â†’ [0.1, -0.3, 0.5, 0.2] 4 angka sekaligus! Lebih kaya maknanya

ğŸ² Darimana Angka-angka Ini?

Awalnya random (acak)! Kita generate pakai rumus distribusi normal (Gaussian) dengan rata-rata 0 dan standar deviasi 0.1

Nanti saat training, AI akan "belajar" nilai terbaik untuk angka-angka ini.

Di awal, semua knop ini kita puter ngasal (Random). Makanya output suaranya (tebakan AI) pasti ancur parah awalnya.

Loss = seberapa bego si AI. Makin kecil, makin pinter!

Rumus: loss = -log(probabilitas jawaban benar)

Kalo AI yakin 90% (padahal bener) â†’ loss kecil. Kalo cuma yakin 1% â†’ loss gede!

Python Fungsi generate 
def generate(start_word):
    current = tokenize(start_word)
    result = start_word
    
    for _ in range(50):  # Max 50 karakter
        input_seq = current[-3:]  # 3 huruf terakhir
        probs = forward(input_seq)
        next_char = chars[argmax(probs)]
        
        if next_char == '.':
            break
            
        result += next_char
        current.append(stoi[next_char])
    
    return result
